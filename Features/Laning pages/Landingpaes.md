## Overview
The Artist Landing Page Generator automatically creates a professional,
fully-branded one-page website for any musician using structured inputs
provided through the MusicAI App.

This system is designed to:
- Increase artist credibility
- Improve conversion for booking managers
- Provide an EPK-quality digital presence
- Simplify promo, music links, and fan engagement

---

## Core Components

### 1. Identity Intake Engine
Captures artist essentials:
- Stage name
- City / origin
- Genre + subgenre
- Style keywords (vibe scan)
- Influences
- Short + long bio inputs
- Social links
- Music links (Spotify, Apple, SoundCloud, YouTube, etc.)

### 2. Theme Generator (AI-Driven)
MusicAI provides multiple aesthetic templates:
- Neon Anime
- Black & Gold Minimal
- Lofi Aesthetic
- Afrofuturist Glow
- R&B Smooth
- Streetwear Bold
- Custom (premium)

Each template includes:
- Color palette
- Typography set
- Header banner prompt
- Social tile variants

### 3. Auto Bio System
MusicAI generates:
- 2-line intro
- 1-paragraph EPK bio
- Long-form bio for press kits

All bios are optimized for SEO and brand tone.
