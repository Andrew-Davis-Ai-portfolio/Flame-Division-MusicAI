## Reporting Security Issues

If you discover a security vulnerability:

1. DO NOT publicly disclose it.  
2. Email: **flamedivision.ai@proton.me**  
3. Provide:
   - Description  
   - Steps to reproduce  
   - Potential impact  

We will respond within 72 hours.

---

## Supported Security Practices

- Ethical use only  
- No malicious testing  
- No exploitation of the system  
- No bypassing or disabling safety mechanisms  

---

## Disclosure

All responsible disclosures will be acknowledged once resolved.
