## 1. System Purpose
FlameBeat-Command is the core rhythmic engine of the ShadowSoundProtocol suite.  
Its mission: generate industry-ready drum patterns, percussive textures, and rhythmic frameworks that align with the artist’s true identity and sonic fingerprint.

Designed for:
- Producers  
- Songwriters  
- Hook builders  
- AI-powered studio flows  
- Mobile musicians using limited hardware  

---

## 2. Core Features

### **Feature A — Pattern Generator**
Creates dynamic, evolving drum sequences with:
- Kick logic  
- Snare placement intelligence  
- Hi-hat swing variations  
- Bounce-mapping for modern trap, drill, R&B, afro, EDM  

### **Feature B — Style Lock**
User can lock in:
- Tempo  
- Bounce setting  
- Groove family  
- BPM emotion category (rage, soulful, hyper, cinematic, minimal)  

The engine will NOT generate outside these boundaries.

### **Feature C — Dynamic Layering**
Adds:
- Percussion accents  
- Reverse hits  
- Transition risers  
- Fill-ins every 4/8 bars  
- Adaptive automation  
All in perfect sync with artist style.

### **Feature D — FlameSync**
Optional mode that syncs beat patterns to:
- Melody frameworks  
- Hook patterns  
- Vocal cadence suggestions  

---

## 3. Command Syntax

### **Generate Standard Pattern**

/beat standard bpm:140 mood:dark bounce:trap

### **Generate Drill Bounce**

/beat drill bpm:144 pattern:syncopated hats:triplet

### **Generate R&B Slow Flow**

/beat rnb bpm:92 style:smooth hats:light_percussion

### **Generate Percussion-Only Layer**

/beat percussion layer:intense bpm:128

### **Lock Artist Style**

/beat style-lock artist:“Powerman Analyst”

## 4. Output Package
Each generation returns:
- **Beat Description** (what was built)  
- **Timing Grid** (kick, snare, hats)  
- **Suggested Melody Mood**  
- **Mixing Notes**  
- **Optional Automation** (FX lanes)
